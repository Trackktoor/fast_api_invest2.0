# Description

## Requirements

Package requirements in "__requirements.txt__"

Other requirements:
    
    - python3.11

## Docker

Run in Docker:

    # build the container ignoring caching
    $ docker-compose build --no-cache

    # run the container
    $ docker-compose up

    # run the container in daemon mode
    $ docker-compose up -d

    # stop the container and removing
    $ docker-compose down
